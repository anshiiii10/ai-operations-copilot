# AI Operations Copilot

## Overview

AI Operations Copilot is an AI-powered application that helps business and operations teams analyze datasets, generate insights, visualize KPIs, and answer business questions using natural language.

## Planned Features

- Upload CSV/Excel files
- Automatic data cleaning
- KPI dashboard
- AI-generated business insights
- Natural language Q&A
- Downloadable reports

## Tech Stack

- Python
- Pandas
- Streamlit
- SQL
- Plotly
- Git
- GitHub
